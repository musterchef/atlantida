Formule Chiave

Haversine (distanza tra due coordinate)

R = 6,371,000 m
a = sin²(Δlat/2) + cos(lat1) · cos(lat2) · sin²(Δlon/2)
d = R · 2 · atan2(√a, √(1-a))

Bearing (direzione di marcia)

x = sin(Δlon) · cos(lat2)
y = cos(lat1) · sin(lat2) - sin(lat1) · cos(lat2) · cos(Δlon)
bearing = atan2(x, y)  →  normalizzato 0–360°

Time Warp

durata_reale_max = max(tutte le tappe)  // es. 40292s (Tappa 3)
scale = 600.0 / durata_reale_max

Per ogni tappa:
  target_duration = durata_reale × scale
  td_time[i] = elapsed[i] × (target_duration / durata_reale)

Interpolazione meteo (lineare fra campioni 15-min)

Per ogni frame:
  progress = frame_i / (n_frames - 1)
  real_timestamp = start_ts + progress × real_duration
  
  // Trova i due campioni meteo che racchiudono real_timestamp
  frac = (real_ts - meteo[j].ts) / (meteo[j+1].ts - meteo[j].ts)
  
  // Campi continui: lerp
  temperature = meteo[j].temp + (meteo[j+1].temp - meteo[j].temp) × frac
  
  // Campi discreti: nearest
  weather_code = meteo[j].code  if frac < 0.5  else  meteo[j+1].code

Esecuzione Completa

# 1. Attiva virtual environment
.venv\Scripts\Activate.ps1

# 2. Pipeline principale (GPX → CSV per TD)
python src/desnivel_gpx_to_td.py

# 3. Meteo (richiede output del passo 2)
python src/weather_fetch.py

# 4. GoPro mapping (richiede ffprobe installato)
python src/gopro_mapper.py

# 5. GoPro tracce GPS
python src/gopro_tracks.py

# 6. POI discovery (lungo, usa API Wikipedia)
python src/poi_discovery.py
